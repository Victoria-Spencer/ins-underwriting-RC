### 项目概述（Project Overview）
ins-underwriting-RC 是面向保险行业的智能核保（Underwriting）解决方案，聚焦于解决传统人工核保效率低、决策标准不统一、风险识别能力不足等痛点。项目基于 JDK 21 构建核心服务层，通过智能体（Agent）作为核心驱动引擎，提供标准化、智能化的核保决策能力，并开放便捷的调用接口支持 Python 生态集成，适用于寿险、财险、健康险等多险种的核保场景。
### 技术栈
SpringBoot + SLF4J + Logback
### 核心目标
• 自动化解析投保资料（病历、体检报告、财务证明等）  
• 融合规则引擎, Python自动化处理 与 AI 模型实现精准风险评估   
• 输出可解释的核保决策结论（承保 / 加费 / 除外 / 拒保）    
• 适配保险业务系统轻量化对接  
### 关键技术说明
• JDK 21：项目核心服务基于 JDK 21 的 LTS 版本开发，充分利用其虚拟线程（Virtual Threads）特性提升高并发核保请求的处理效率，同时兼容密封类、Record 类型等语法简化核心逻辑开发  
• 智能体引擎：项目核心组件，整合 “规则引擎 + AI 模型 + 业务知识图谱”，具备自主解析资料、评估风险、生成决策的能力，且支持规则 / 模型的热更新
